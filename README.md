# Samuni
Samuni is a high-skill, no-grid match game built for people who like a little controlled chaos.
Colored orbs drift freely in a physics playground, and you link them in real time to trigger attacks, energy surges, coin gains, base upgrades, and evolution paths. No timers. No rush. Just pure thinking, strategy, and flow.
Red hits. Blue powers. Yellow earns. Green builds.
Purple is you.
Everything else is noise trying to get in your way.
Swipe for quests. Swipe for upgrades.
Climb seasons. Evolve your star.
All on a single seamless screen.
If you want random luck, play a slot machine.
If you want skill, welcome to Samuni.
